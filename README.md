# Game of Thrones Website

## Опис
Це навчальний вебпроєкт, створений для лабораторних робіт.

Сайт присвячений серіалу "Гра престолів" і містить декілька сторінок:
- Головна   
- Про автора
- Про Джорджа Мартіна
- Головні персонажі
- Галерея

## Технології
- HTML5
- CSS3

## Структура проєкту
- index.html — головна сторінка
- about_autor.html — сторінка про автора
- martin.html — сторінка про автора книги
- characters.html — персонажі
- gallery.html — галерея
- style/ — стилі сайту
- images/ — зображення

## Як запустити
1. Відкрити папку проєкту
2. Запустити файл index.html у браузері

## Мета роботи
Опанування роботи з Git, GitHub, гілками, Pull Request та merge conflicts.
